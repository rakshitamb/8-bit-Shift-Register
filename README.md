# 8-bit-Shift-Register
A shift register is a sequential digital circuit used to store and transfer data. In this project, an 8-bit shift register is designed using Verilog HDL

Overview

This project implements an 8-bit shift register using Verilog HDL. The shift register can move data bits either to the left or right and supports Serial-In Parallel-Out (SIPO) and Parallel-In Serial-Out (PISO) operations. The functionality of the design is verified through ModelSim simulation.

Features
8-bit sequential shift register design
Supports shift-left and shift-right operations
Implements SIPO and PISO data transfer
Developed using Verilog Hardware Description Language (HDL)
Verified using ModelSim waveform simulation

Description
A shift register is a sequential digital circuit used for storing and transferring binary data. In this project, an 8-bit shift register is designed using Verilog HDL. The register shifts data based on clock signals and control inputs, enabling movement of bits either to the left or right. The design also demonstrates data conversion between serial and parallel formats, and its functionality is verified using a testbench simulation in ModelSim.

Learning Outcomes
Understand the working of shift registers in digital systems
Design sequential circuits using Verilog HDL
Implement shift-left and shift-right operations
Learn the concepts of SIPO and PISO shift registers
Verify digital designs using ModelSim simulation
